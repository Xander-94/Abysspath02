# Settings 子页面清单

## 数据管理控制台 (DataManagementConsole)
### 功能
- 数据清理
  - 本地缓存清除
  - 临时文件清理
  - 历史记录清理
- 数据导出
  - 历史数据导出(JSON/CSV)
  - 自定义导出范围
  - 导出进度跟踪
- 账户管理
  - 账户注销
  - 数据备份
  - 恢复出厂设置
### 布局
- 危险操作红色强调
- 操作确认对话框
- 进度显示条

## 隐私控制面板 (PrivacyControlPanel)
### 功能
- 数据共享
  - 改进计划参与
  - 匿名数据收集
  - 使用统计分享
- 个性化设置
  - 推荐偏好
  - 广告偏好
  - 消息推送设置
### 布局
- 开关列表布局
- 详细说明提示
- 分组展示
### 交互
- 开关状态切换
- 更多设置展开
- 帮助信息查看
