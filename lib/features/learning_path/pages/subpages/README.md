# Learning Path 子页面清单

## 路径调整控制台 (PathAdjustmentConsole)
### 功能
- 路径调整
  - 拖拽阶段顺序
  - 自定义时间分配
  - 阶段内容编辑
- 冲突检测
  - 时间过载提示
  - 依赖关系检查
  - 资源冲突预警
### 交互
- 实时计算
  - 总时长显示
  - 进度预测
  - 资源消耗估算
- 拖拽操作
  - 阶段排序
  - 时间调整
  - 内容重组

## 阶段详情弹窗 (StageDetailDialog)
### 触发方式
- 点击路径阶段卡片
### 功能
- 阶段信息
  - 目标说明
  - 学习要求
  - 评估标准
- 资源管理
  - 添加备注
  - 上传附件
  - 链接资源
### 布局
- 右侧滑出面板
- 可调整宽度
- 支持手势关闭
