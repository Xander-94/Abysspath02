# Learning Path 页面清单

## 学习路径页 (LearningPathPage)
- 路径概览
  - 学习阶段可视化
  - 当前进度展示
  - 里程碑标记
- 任务管理
  - 待完成任务列表
  - 任务优先级排序
  - 任务完成状态
- 进度控制
  - 学习进度条
  - 时间规划
  - 完成度统计
- 学习资源
  - 课程列表
  - 学习材料
  - 推荐资源
- 学习计划
  - 每日任务
  - 周计划
  - 月目标

## 当前实现的功能

*   **LearningPathPage (`learning_path_page.dart`)**:
    *   作为学习路径模块的主入口页面。
    *   包含一个**历史记录抽屉 (Drawer)**，用于显示和切换用户的学习路径。
    *   提供一个**聊天式界面**，用户可以通过输入学习目标来创建新的学习路径。
    *   显示当前选中学习路径的对话消息。
    *   在创建新路径成功后，会显示一条包含**"查看详情"按钮**的确认消息。
    *   实现了历史路径列表的**乐观更新**，确保创建后即时显示在列表中。
    *   处理了从数据库加载历史路径和消息的逻辑。
*   **LearningPathDetailPage (`learning_path_detail_page.dart`)**:
    *   用于展示单个学习路径的详细信息。
    *   顶部显示路径的标题、描述、预估时长和目标。
    *   包含两个 Tab 视图：**知识图谱** 和 **节点列表**。
    *   知识图谱视图使用 `graphview` 库可视化展示路径的节点和关系。
    *   节点列表视图以列表形式展示所有节点及其信息。
    *   实现了从 API 获取并展示完整学习路径数据的逻辑。
